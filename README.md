# ubermanu.github.io

This is the source code for my personal website. It is built using [SvelteKit](https://kit.svelte.dev/) and [TailwindCSS](https://tailwindcss.com/).

## Dev

Clone the repository and install the dependencies.

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
